# One-Week MVP

## Day 1

- Build CEF app bundle.
- Verify the CEF Views window opens.
- Confirm WebUI resources load from the bundle.

## Day 2

- Finish tab add/switch/close flows.
- Add URL bar navigation.
- Add JSON session persistence for tabs.

## Day 3

- Replace terminal-lite with xterm.js.
- Wire resize events to `PtySession::Resize`.
- Add basic shell integration markers.

## Day 4

- Store command blocks.
- Show command status and exit code.
- Add replay for recent commands.

## Day 5

- Add permission UI.
- Show risk reasons for high-risk commands.
- Add allow-once and allow-session decisions.

## Day 6

- Add browser context tool.
- Add file diff preview for agent writes.
- Add model provider adapter.

## Day 7

- Polish the end-to-end demo.
- Package a macOS app bundle.
- Record demo scripts and known limitations.
