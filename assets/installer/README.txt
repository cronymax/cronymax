cronymax
========

Thank you for downloading cronymax!

INSTALLATION
------------
Drag cronymax.app to the Applications folder using the alias in this window.


FIRST LAUNCH — GATEKEEPER NOTICE
---------------------------------
Because cronymax is not yet signed with an Apple Developer certificate, macOS
will show a warning when you open it for the first time:

  "cronymax.app cannot be opened because Apple cannot check it for
   malicious software."

To open the app:

  1. Right-click (or Control-click) cronymax.app in /Applications
  2. Choose "Open" from the context menu
  3. Click "Open" in the dialog that appears

You only need to do this once. After the first launch, cronymax opens normally.

ALTERNATIVE (Terminal)
----------------------
If you prefer the command line, run:

  xattr -d com.apple.quarantine /Applications/cronymax.app

Then double-click as normal.


SYSTEM REQUIREMENTS
-------------------
  - macOS 14.5 (Sonoma) or later
  - Apple Silicon (arm64) — use the arm64 build
  - Intel Mac (x86_64) — use the x86_64 build
